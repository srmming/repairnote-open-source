# RepairNOTE 老板使用说明

## 1. 登录

打开系统网址，输入账号和密码登录。默认测试账号是：

- 账号：`ming`
- 密码：`123456`

正式试用前建议在“员工”页面新增真实员工账号，并给老板账号勾选管理员。

## 2. 开一张维修单

进入“维修单”，点击“新增”。

填写顺序建议：

1. 先输入客户电话。
2. 如果系统出现已有客户，点一下客户卡片复用资料。
3. 如果是新客户，填写客户姓名和电话。
4. 填写品牌、型号、IMEI、维修内容。
5. 选择或输入服务项目、配件项目和价格。
6. 填写报价、优惠、订金。
7. 点击底部“创建”。

创建成功后，维修单会出现在维修单列表里。

## 3. 打印给客户

在新增或编辑维修单页面右侧价格区域，可以打印：

- “打印小票”：适合热敏小票机。
- “打印 A4”：适合普通打印机。

打印内容会包含客户信息、设备信息、维修内容、费用、订金、待收款和维修进度二维码。

## 4. 客户扫码查进度

每张维修单都会生成一个二维码进度页。客户扫码后可以看到：

- 维修单号
- 当前状态
- 客户姓名
- 设备品牌和型号
- 维修内容
- 更新时间
- 待收款金额

内部备注不会显示给客户。

## 5. 修改维修状态

在维修单列表里可以直接修改状态，也可以进入维修单详情修改。

常用状态：

- 预定
- 预定已到货
- 待开始
- 维修中
- 完成
- 已取走
- 关闭

建议每天先处理“待开始”和“维修中”的单。

## 6. 管理客户

进入“客户”页面，可以新增、编辑、搜索客户。

如果客户已有维修单，系统不允许直接删除，避免历史维修记录丢失。

客户等级支持：

- 普通
- VIP
- 超级 VIP
- 黑名单

黑名单客户开单时会有提醒。

## 7. 管理品牌、型号、服务和配件

常用手机品牌、型号、维修服务和配件都可以在后台维护。

建议先把门店最常用的项目录进去，比如：

- 换屏
- 换电池
- 充电口
- 摄像头玻璃
- 钢化膜
- 手机壳

服务可以设置价格，开单时点选后会自动加入费用项目。

## 8. 看报表

进入“报表”页面，可以查看：

- 收入
- 维修单数量
- 平均客单价
- 未收款金额
- 热门品牌/型号
- 热门服务
- 状态分布

可以按今天、本周、本月切换。

## 9. 设置打印信息

进入“设置”页面，可以设置：

- 联系电话
- 税率
- 打印语言
- 是否隐藏出票人
- 维修条款
- 保修条款
- 预定条款

这些内容会影响打印单据。

## 10. 备份和恢复

进入“备份”页面：

- 系统每天第一次保存数据时，会自动留一份历史备份。
- 点击“立即生成备份”可以手动留一份当前数据。
- 历史备份列表里可以点击“恢复”，恢复前系统会先自动保存一份安全备份。
- 点击“下载当前备份”或历史备份里的“下载”，可以把 ZIP 压缩包保存到自己电脑。
- 点击“上传备份文件”可以选择系统导出的 ZIP 压缩包或 JSON 文件并恢复。
- “高级导入”里仍然可以导出数据库 JSON、粘贴 JSON 后导入。
- 如果以前用过旧 localStorage 版本，可以点击“导入旧 localStorage”迁移旧数据。
