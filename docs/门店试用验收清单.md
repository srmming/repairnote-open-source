# RepairNOTE 门店试用验收清单

## 已覆盖页面

| 页面 | 当前要求 | 验收证据 |
| --- | --- | --- |
| 登录页 | 账号密码登录、失败提示、中西语切换 | `npm run smoke` |
| 维修单列表 | 搜索、筛选、分页、状态修改、进入详情 | `npm run smoke` |
| 新增/编辑维修单 | 客户复用/新增、品牌型号、维修内容、价格项目、订金、优惠、保存 | `npm run smoke`、`npm run smoke:mobile` |
| 打印 | A4、小票、二维码进度页链接 | `npm run smoke` |
| 客户管理 | 新增、编辑、删除限制、搜索、分页 | `npm run smoke` |
| 品牌/型号 | 新增、编辑、删除限制、搜索 | `npm run smoke` |
| 配件/服务/属性 | 新增、编辑、删除、搜索、分页 | `npm run smoke` |
| 员工 | 新增、编辑、删除限制、管理员保护 | `npm run smoke` |
| 报表 | 收入、维修单数量、客单价、未收款、排行、状态分布 | `npm run smoke` |
| 设置 | 联系电话、税率、打印语言、条款 | `npm run smoke` |
| 备份 | 导出数据库 JSON、导入 JSON、坏 JSON 提示、旧 localStorage 导入入口 | `npm run smoke` |
| 客户扫码进度页 | 公开 token 页面、手机端可读 | `npm run smoke`、`npm run smoke:mobile` |
| 手机端 | 底部导航、列表卡片、新增维修单、公开进度页 | `npm run smoke:mobile` |
| 真实用户逐页巡检 | 桌面/手机逐页截图、关键入口、横向溢出、控制台错误 | `npm run audit:real-user` |

## 试用前必须继续处理

- 给真实门店配置店名、电话、打印条款、默认服务/配件价格。
- 用一份旧数据 JSON 做一次真实导入演练，确认客户、维修单、品牌型号都能落库。
- 老板使用说明已补：`docs/老板使用说明.md`。
- 长期多员工并发使用前，把高频操作继续改为按资源/单条记录保存；当前版本已有数据版本保护，可避免旧快照静默覆盖新数据。
